🌌 FrostByte - Options de Déploiement

🔧 Option 1: GitHub Pages (Repository Public)

Étape 1: Rendez votre repository public
→ GitHub.com → FrostByte → Settings → Danger Zone → Change visibility → Public
Étape 2: Activez GitHub Pages
→ Settings → Pages → Source: GitHub Actions → Save
🔗 GitHub Settings

🚀 Option 2: Netlify (Recommandé)

Étape 1: Connectez-vous à Netlify
→ Créez un compte avec GitHub
Étape 2: Déployez votre site
→ New site from Git → GitHub → FrostByte → Deploy
🚀 Déployer sur Netlify

⚡ Option 3: Vercel

Import depuis GitHub:
→ Connectez GitHub → Import FrostByte → Deploy
⚡ Déployer sur Vercel
